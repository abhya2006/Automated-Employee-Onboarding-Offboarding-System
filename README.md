# Automated Employee Onboarding & Offboarding System

## Overview
This project automates employee onboarding and offboarding using ServiceNow Creator Studio.

## Features
- HR Verification
- Manager Approval
- IT Asset Allocation
- Facilities Setup
- Security Access Control
- Automated Workflow using Playbook

## Tools Used
- ServiceNow
- Creator Studio
- Playbook Automation

## Workflow
Employee onboarding request automatically creates tasks for HR, Manager, IT, Facilities and Security teams.
